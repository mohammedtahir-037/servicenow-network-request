# Automated Network Request Management in ServiceNow

## 📌 Project Overview

This project automates network request handling using ServiceNow. It includes request submission, approval workflow, task creation, and completion tracking.

## 🚀 Features

* Service Portal request submission
* Automated approval flow
* Record creation in Network Database
* Status tracking (New → Approved → Completed)
